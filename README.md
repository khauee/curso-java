# ☕ Java Completo: Programação Orientada a Objetos + Projetos

Este repositório contém o registro da minha jornada de aprendizado no curso de **Java Completo e Orientado a Objetos**, ministrado pelo Prof. Nélio Alves. O foco principal é dominar a linguagem Java sob a ótica da POO e as principais tecnologias do ecossistema moderno.

## 🚀 Objetivo do Repositório
O objetivo é consolidar os fundamentos de programação, desde a lógica básica até o desenvolvimento de sistemas complexos utilizando o ecossistema Spring Boot e persistência de dados.

## 🛠️ Tecnologias e Ferramentas
* **Linguagem:** Java (versão LTS)
* **Frameworks:** Spring Boot, JPA/Hibernate, Spring Data (JPA & MongoDB)
* **Banco de Dados:** SQL (JDBC) e NoSQL (MongoDB)
* **Gerenciamento de Dependências:** Maven
* **Versionamento:** Git & GitHub
* **Interface Gráfica:** JavaFX

## 📚 Tópicos Cobertos
- [x] **Lógica de Programação:** Algoritmos, sintaxe e estruturas de controle.
- [x] **POO Fundamental:** Classes, atributos, métodos, construtores e encapsulamento.
- [ ] **POO Avançada:** Herança, polimorfismo, interfaces e classes abstratas.
- [ ] **Estrutura de Dados:** Listas, arrays, enumerações e composição.
- [ ] **Tratamento de Exceções:** Robustez e segurança no código.
- [ ] **Programação Funcional:** Expressões Lambda e Stream API.
- [ ] **Persistência de Dados:** Integração com bancos SQL e NoSQL.
- [ ] **Web Services:** Desenvolvimento de APIs REST com Spring Boot.

## 📁 Estrutura do Projeto
O projeto está organizado em **Módulos** para facilitar o estudo:

* **fundamentos:** Contém toda a base lógica e introdução à Orientação a Objetos (Pacotes separados por assunto).
* **projeto-xadrez:** (Em breve) Aplicação completa de jogo de xadrez no console.
* **web-services:** (Em breve) API RESTful com Spring Boot e Banco de Dados.
